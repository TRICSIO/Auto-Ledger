# **App Name**: Auto Ledger

## Core Features:

- Expense Logging: Allow users to log vehicle expenses (fuel, maintenance, repairs, etc.) with details like date, amount, and description.
- Expense Categorization: Categorize expenses automatically and manually.
- Combined Expense View: Present a combined expenses view for all vehicles.
- Vehicle Identification: Allow users to specify each vehicles make, model, year, and trim, engine type, drive type, transmission, VIN, license plate, mileage.
- Vehicle Specific Expense View: Vehicle specific expenses, displayed in tabular and pie chart formats.
- AI Recall Tool: AI-powered tool for vehicle recall checking based on entered vehicle details (make, model, year). The AI reasons about the best time to notify a user about vehicle recalls, avoiding unnecessary alerts when recall is already known.
- Maintenance Schedule: Allow users to track routine maintenance tasks with mileage and date prompts. Track tire rotations, oil changes and other scheduled tasks. Prompt them at defined mileage intervals.

## Style Guidelines:

- Primary color: Blue-gray (#5F7A8F) to evoke reliability and efficiency.
- Background color: Light gray (#EAEFF2), creating a clean and professional backdrop.
- Accent color: Teal (#468999) to highlight key actions and information.
- Font pairing: 'Space Grotesk' (sans-serif) for headings, 'Inter' (sans-serif) for body text. 
- Use a consistent set of glyph-style icons throughout the app, representing expense categories, maintenance tasks, and vehicle identification elements.
- A clean, card-based layout with intuitive navigation.
- Subtle transitions and animations when navigating between views and adding new expenses or maintenance records.
